【使い方】
1. ZIPを右クリックして「すべて展開」します。
2. 展開した margherita_game_complete フォルダをVS Codeで開きます。
3. index.htmlを右クリック → Open with Live Server。
4. assetsフォルダの画像を本物の写真に差し替える場合は、名前を変えずに上書きしてください。
   margherita.png
   konbu.png
   okayu.png

【入っている機能】
・3匹が別の高さを走る
・上下に弾む走行アニメーション
・スピード線
・おやつ獲得時の「+点」表示
・最高スコア保存
・音のオン／オフ
・スマホ対応
・ホーム画面追加対応
